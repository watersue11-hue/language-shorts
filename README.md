# 무료 다국어 학습 숏츠 생성기 - 사진 스타일 카드형 UI 버전

사용자가 보낸 레퍼런스 이미지처럼 영상 화면을 바꾼 버전입니다.

## 디자인 수정

- 뒤쪽 A / 일본어 / i / 중국어 같은 언어 문자 박스 제거
- 배경은 흰색/연회색 위주로 단순화
- 메인색은 파랑 하나만 사용
- 글자는 검정/진회색/파랑만 사용
- 상단 `오늘의 표현` 제목 박스 유지
- 메인 한국어 문장 유지
- 외국어 표현은 큰 흰색 카드 안에 파란색으로 표시
- `못 들었을 때` 설명 유지
- 하단 파란 웨이브만 장식으로 유지

## 표현 개수

- 쇼츠: 1개~10개
- 롱폼: 1개~50개

## Streamlit Cloud 배포

1. GitHub에 이 폴더 안의 파일을 전부 업로드합니다.
2. Streamlit Community Cloud에서 repo를 연결합니다.
3. Main file path를 `app.py`로 설정합니다.
4. Reboot 또는 Deploy를 누릅니다.

Secrets 설정은 필요 없습니다.
